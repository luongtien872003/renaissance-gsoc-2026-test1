# RenAIssance GSoC 2026 – Test I

This repository contains my work for the RenAIssance prospective student evaluation test (Test I: OCR of printed sources).

## Goal
Build an OCR pipeline for early modern printed sources, focusing on main text detection and recognition, and optionally using an LLM as a late-stage correction step.

## Current pipeline
- PDF page rendering
- Preprocessing
- Main text / line detection
- OCR inference
- Text normalization
- Evaluation with CER

## Repository status
Work in progress. I am currently improving line segmentation, OCR quality, and late-stage correction.
